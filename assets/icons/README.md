# Tray Icons

This directory contains tray icons for different platforms.

## Required Icons

### macOS
- `tray_icon.png` - 16x16 PNG with transparency
- `tray_icon@2x.png` - 32x32 PNG with transparency

### Windows
- `tray_icon.ico` - 16x16 and 32x32 ICO format

## Notes
- Icons should be simple and monochrome for best system tray appearance
- macOS requires @2x retina version
- Windows ICO should contain multiple sizes
